不使用ajax方式提交表单不跳转页面

发表于 2015-07-07

  前些天碰到了一个Excel文件上传解析的功能要做，需要实现在提交Excel文件后，依然驻留在本页面。当时没有想到要怎么实现，除了ajax方式外，但是当时不知怎么我脑子抽风不想使用ajax，还好让我找到一种原始的实现方式。实现思路就是在页面写一个空的iframe，再写一个span空标签，原始的表单action为空，target为上述的iframe。代码片段如下：

<form id="attachmentForm" 
    action="${ctx}/animalepidemic
            /basiciiformation/epWarehouse/importExcel?id=${epWarehouse.id}" 
    method="post" enctype="multipart/form-data"
    target="attachment_iframe" 
    style="display:none;width:0px;height:0px;">
    <span id="attachmentContent"></span>
    <!--  <input type="hidden" id="attachmentNum" name="attachmentNum"/> -->
</form>
<div id="myModal" class="modal hide fade" tabindex="-1"
    role="dialog" aria-labelledby="myModalLabel" aria-hidden="true">  
      <div class="modal-header">  
         <button type="button" class="close"
              data-dismiss="modal" aria-hidden="true">×</button>  
         <h3 id="myModalLabel">选择文件</h3>  
     </div>  
     <div class="modal-body">  
         <form  method="post" target="attachment_iframe">                 
          <input id="filesubmit" type="file" style="display:inline-flex;"
               name="importExcel" id="importExcel">                  
        </form> 
      </div>  
     <div class="modal-footer">  
         <button class="btn" data-dismiss="modal" aria-hidden="true">取消</button>  
         <shiro:hasPermission name="animalepidemic:basiciiformation:epWarehouse:edit">
             <button class="btn btn-primary" data-dismiss="modal"
                  aria-hidden="true" onclick="comfirm()">
                 确认
             </button>
         </shiro:hasPermission>  
     </div>  
 </div> 
<script type="text/javascript">    
    function comfirm(){
        var af = $('#filesubmit');
        //检查是否选择了文件
        if(af.val()=='') {
            $.alert("提示",'请选择要上传的文件');
            return;
        }
        //将file移动至上传表单
        $('#attachmentContent').empty();
        $('#attachmentContent').append(af);
        //复制一个file放至原处
        $('#filesubmit').append(af.clone().attr('value',''));
        //$('#attachmentText'+n).attr('value','');
        //修改属性
        af.attr('id','');
        //其他表单
        //$('#attachmentNum').val(n);
        $('#attachmentForm').submit();
        //$('#filesubmit').submit();                    

    }
</script>
