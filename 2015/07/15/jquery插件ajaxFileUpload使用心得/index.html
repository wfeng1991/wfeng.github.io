jquery插件ajaxFileUpload使用心得

发表于 2015-07-15

  今天需要做一个异步excel文件上传并且在页面以table形式显示出来。我在网上找了半天发现Jquery的一个文件上传插件ajaxFIleUpload很好用。于是我就下载了一个试了下，确实使用起来很简单。但是有个问题，就是后台（我用的servlet）返回的数据是String类型时，而前端设置的dataType设置为’json’时，即使上传成功也不会调用success回调函数。同样也没办法解析后台传回来的数据。后来发现后台传回来的字符串不符合json格式，ajaxFileUpload.js中执行以下方法是后出错：

uploadHttpData: function( r, type ) {
        var data = !type;
        data = type == "xml" || data ? r.responseXML : r.responseText;
        // If the type is "script", eval it in global context
        if ( type == "script" )
            jQuery.globalEval( data );
        // Get the JavaScript object, if JSON is used.
        if ( type == "json" )                   
           eval( "data = " + data );
        // evaluate scripts within html
        if ( type == "html" )
            jQuery("<div>").html(data).evalScripts();

        return data;
    }

  这里就有两个选择要么该ajaxFileUpload.js源代码，要么改dataType为text。我觉得还是尽量采取第二种方法，因为这个插件可能还有别人在使用，要注意的是后者需要自己解析传回来的字符串（字符串可能会跟后台传的原始字符串有点不同，我的是被包裹在一个pre标签中），但也没那么麻烦。以下是我的代码：

function comfirm(){
    var af = $('#filesubmit');
    //检查是否选择了文件
    if(af.val()=='') {
        alert('请选择要上传的文件');
        return;
    }                
    var url = '${ctx}/projsub/warehouse/Warehouse/importExcel?id=${warehouse.id}';
    $.ajaxFileUpload({
              url:url,//服务器端程序
              secureuri:false,
              fileElementId:'filesubmit',//input框的ID
              dataType: 'text',//返回数据类型
              beforeSend:function(){//上传前需要处理的工作，如显示Loading...
          },
          success: function (data, status){//上传成功
            var  epWarehousedtsTpl = $("#epWarehousedtsTpl").html()
                                    .replace(/(\/\/\<!\-\-)|(\/\/\-\->)/g,"");
            data= data.substring(data.indexOf("[") , data.lastIndexOf("]")+1);
            var epWarehousedtsRowIdx = $('#epWarehousedtsList tr').length;
            data =  eval('('+data+')'); 
            for (var i=0; i<data.length; i++){
                addRow('#epWarehousedtsList', epWarehousedtsRowIdx,
                         epWarehousedtsTpl, data[i]);
                epWarehousedtsRowIdx = epWarehousedtsRowIdx + 1;
            }
          },

        });
    //只允许导入一次
    $('#import').hide();
    //$('#displayform').empty();
    //$('#displayform').append(af.clone().attr('value',''));

}

  另外生成table时使用的是Mustache。Mustache是基于JavaScript实现的模版引擎，类似于JQuery Template，但是这个模版更加的轻量级，语法更加的简单易用，很容易上手。我对这个暂时还不是很熟，下次有了心得再写点。
